Website ini dikembangkan sebagai proyek Ujian Akhir Semester Mata Kuliah Pemrograman Web 1 pada Program Studi Sistem Informasi, Fakultas Ilmu Komputer, Universitas Pamulang.

Tentang Mata Kuliah

Mata kuliah ini mempelajari konsep dasar perancangan dan pengembangan aplikasi berbasis web. Fokus pembelajaran mencakup penyusunan struktur halaman menggunakan HTML, perancangan tampilan dan tata letak antarmuka menggunakan CSS, serta penerapan fungsi interaktif pada sisi klien (client-side) menggunakan JavaScript untuk menciptakan pengalaman pengguna yang lebih dinamis dan responsif.
